# Product screenshots

Drop real screenshots of the Orb application here (`.png`, `.jpg`, `.webp`).
They appear automatically in the "Straight from the product" section on the
landing page, sorted by filename — prefix with `01-`, `02-`, … to control
order. Captions are derived from the filename:

    02-ward-overview.png  →  "Ward overview"

While this folder has no images, the section renders nothing — same rule as
the proof band: no real artifact, no section. Please only add genuine
screenshots of the actual application; the whole point of this section is
that it is not a mockup.
